# VaultX Oripa Platform

GitHub Pagesにそのまま配置できる、Solana入金からガチャ、入手品、発送依頼またはポイント還元までのフローを再現した静的Webアプリです。

## ファイル
- `index.html`
- `style.css`
- `script.js`

## できること
- Solanaウォレット接続風UI
- SOL入金 → ポイント反映
- ポイントでガチャ
- Proof Hash付きの入手品保存
- 発送依頼
- ポイント還元
- 誰でもBOX出品できるSeller Studio
- 販売手数料モデル表示

## 注意
これはフロントエンドのプロトタイプです。本番化には決済、法務、本人確認、配送管理、景品表示、確率監査、スマートコントラクト等の実装が必要です。
